# Project Overview

## Research Questions

主要研究问题：教育回报率在数字化时代的变化趋势及其影响因素

## Hypotheses

- **H1**: 数字化技能教育显著提高了个体的收入水平
- **H2**: 教育回报率存在代际差异，年轻群体的教育回报更高
- **H3**: 技术进步加剧了高技能和低技能劳动者之间的收入不平等

## Datasets

- **主要数据集**:
  - 中国家庭追踪调查 (CFPS) 2018-2022
  - 样本量: ~40,000 个体
  - 变量: 教育水平、收入、数字技能、职业信息

- **补充数据集**:
  - 国家统计局年度数据
  - 教育部统计年鉴
  - 互联网普及率数据

## Methodology

- **基础模型**: Mincer 方程扩展形式
- **计量方法**: OLS + 工具变量法 + 分位数回归
- **稳健性检验**: 替代变量构造 + 样本分割 + 异质性分析

## Key Variables

- **因变量**: 年收入（对数形式）
- **核心自变量**: 教育年限、数字化技能指数
- **控制变量**: 年龄、性别、工作经验、地区、行业
- **工具变量**: 父母教育水平、早期学校质量
