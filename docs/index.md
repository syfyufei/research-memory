## ğŸš€ Getting Started

Research Memory addresses a fundamental challenge in academic research: **context continuity across AI sessions**.

Instead of starting every conversation with "Where did we leave off?", you can:

- **Restore project context instantly** with natural language
- **Maintain searchable research memory** across sessions
- **Never lose important decisions** or experimental insights

### Quick Start

```bash
# Clone or copy to your research project
cp -r research-memory/* your-project/

# Use in Claude Code
"Hey, help me get back up to speed with my research using research-memory"
```
